Space Shooter

This project was developed in Processing using Java, and it's a basic demonstration of what you can achieve with simple math and methods in Processing.

The main purpose behind this project was to incentivize and guide students from the Mathematic Resources class I was tutoring to develop a good final project and have a base to what their projects could look like. In addition to the project source on GitHub, I algo made a serie of videos on youtube giving a quick walkthrough in the code and its systems, which you can check bellow (in Portuguese).
